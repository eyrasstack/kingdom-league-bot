About

Bringing League of Legends data to your Discord community.

Kingdom League Bot is a feature-rich Discord bot that connects your server members' Discord accounts to their Riot Games accounts, enabling instant access to ranked stats, match history, live game tracking, and more — all without leaving Discord.

Players link their accounts through a secure icon-based verification process, then use simple commands to look up stats for themselves or any linked server member. The bot also runs background services for automated match notifications, daily performance awards, and weekly leaderboard recaps.

Built for EUW communities, the bot uses the official Riot Games API and stores only the minimal data needed to operate — Discord IDs and Riot IDs in a local SQLite database.

User Flow

How users link their accounts and interact with the bot.

1

Invite the Bot

A server admin adds the bot to their Discord server. The bot joins and is ready to accept commands from any member.

2

Link Your Riot Account

A user types the link command with their Riot ID (game name and tagline). The bot looks up the account via the Riot API to confirm it exists.

!link YourName#EUW
3

Verify Account Ownership

The bot responds with a specific summoner icon number (one of the free default icons, 1–25). The user must open their League client and change their summoner icon to the requested icon within 10 minutes. This proves the user owns the account.

Example: "Please change your summoner icon to icon #14, then type !verify"

4

Complete Verification

The user types the verify command. The bot checks the Riot API to confirm the summoner icon matches the requested icon. If it matches, the account is linked. The Discord ID and Riot ID are stored in the database.

!verify
5

Use Commands

Once linked, the user can run any command. Commands work on themselves by default, or they can mention another linked server member to look up that player's stats.

!rank
!rank @friend
!history 5
6

Automated Tracking

The bot runs background services that automatically track matches, send notifications when games finish, and post daily/weekly awards — all without user input. Admins configure the notification channel once with !setchannel.

7

Unlink Anytime

Users can unlink their account at any time. This removes their Discord-to-Riot association from the database. They can re-link with a different account if needed.

!unlink

Commands

Full command reference. All commands use the ! prefix.

Account Management

!link name#tagLink your Discord account to your Riot account (starts verification)
!verifyComplete account verification by confirming summoner icon change
!unlinkRemove the link between your Discord and Riot accounts
!meShow your profile overview (rank, level, top champions)
!settingsManage account settings (e.g. toggle flex rank display)

Ranked Stats

!rank [@user]Show ranked stats with peak rank tracking
!lpShow LP gains and losses for the day
!streak [@user]Show current win/loss streak
!tilt [@user]Analyze tilt level based on recent games (0–100 score)
!stats [@user]Detailed stats from last 20 games (KDA, CS, damage, vision)
!leaderboardShow server ranking leaderboard (top 10 players)
!history [@user] [count]Show match history with MVP/Ace badges (up to 10 games)

Player Profile

!profile [@user]Detailed player profile
!mastery [@user]Show top champion masteries
!peakrank [@user]Show peak rank vs current rank with progress
!milestones [@user]Track ranking milestones over time
!champion [@user]Show stats for a specific champion
!best [@user]Best performing champions
!worst [@user]Worst performing champions

Live Game

!livegame [@user]Show active game with teams, ranks, and win probability
!livegamesShow all active games for linked players in the server
!clash [@user]Show Clash tournament info and team details

Social & Comparison

!compare name#tag name#tagCompare stats between two players side by side
!headtohead @userHead-to-head comparison with win rates
!synergy name#tag name#tagAnalyze team synergy in games played together
!opgg [@user]Get a direct link to a player's op.gg profile

Custom Games

Privacy Policy

Transparency about how your data is handled.

Data We Collect

  • Your Discord user ID (numeric identifier)
  • Your Riot ID (game name and tagline)
  • Ranked stats and match data retrieved from the Riot API
  • Custom game results you submit (K/D/A, champion, team)
  • LP snapshots for daily tracking

How Data Is Stored

  • All data is stored in a local SQLite database on the bot's server
  • No cloud databases or third-party storage services are used
  • Database is backed up daily for reliability
  • The bot server is hosted on Oracle Cloud with restricted access

Data We Do NOT Collect

  • We do not collect your Riot account password or login credentials
  • We do not read or store your Discord messages
  • We do not track your activity outside of League of Legends
  • We do not collect personally identifiable information (name, email, etc.)

Your Rights

  • You can unlink your account at any time using !unlink
  • Unlinking removes your Discord-to-Riot association from the database
  • Your data is never sold, shared with third parties, or used for advertising
  • You can request full data deletion by contacting the bot operator

Riot Games API Compliance

This application uses the Riot Games API in accordance with the Riot Games API Terms of Service.

We do not store any data beyond what is necessary to provide the bot's functionality. All game data is retrieved in real-time from the official Riot Games API and displayed only within Discord.

Account verification is performed using the summoner icon method — no passwords or login credentials are ever requested or stored.
